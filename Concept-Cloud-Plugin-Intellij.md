# 概述

使用该方式搭建项目结合规范的代码可以对模块进行任意的组合

假设有`A`，`B`，`C`三个模块

- 我们可以实现`A`，`B`，`C`分别作为一个服务的微服务
- 我们可以实现`AB`合并为一个服务，`C`单独为一个服务的微服务
- 我们可以实现`AC`合并为一个服务，`B`单独为一个服务的微服务
- 我们可以实现`BC`合并为一个服务，`A`单独为一个服务的微服务
- 我们可以实现`ABC`合并为一个单体服务
